## 🔒 Seguridad (Headers A+)

| Cabecera | Protección |
|:---|:---|

| `X-Frame-Options` | Protección anti-clickjacking |
| `X-Content-Type-Options` | Evita sniffing de tipos MIME |
| `X-XSS-Protection` | Filtro XSS en navegadores |
| `Permissions-Policy` | Hardware desactivado (Micro/Cam/Geo) |
